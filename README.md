# M5_Dual_Button
Dual Button Unit Code for the M5StickCPlus1.1

## Parts required
1. M5 Stick C Plus 1.1
2. Dual Button Unit

## Installation
1. Download the code from my site
2. Connect the Dual Button Unit to the M5
3. Upload the code to the M5
